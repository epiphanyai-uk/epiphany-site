# Epiphany AI website with resend email API 📮

🎨 Tech Stack

`React + Vite + Typescript`
`Express`
`Resend Email API`
`pnpm`

🚀 Getting Started

Install pnpm

`npm install -g pnpm`

Install dependencies

`pnpm install`

🌐 Frontend

Run Vite dev server:

`pnpm dev`

Frontend runs at:

<http://localhost:5173>

🤖 Backend (Email Server)

Run Express server:

`npx tsx api/server.ts`

Backend runs at:

<http://localhost:3001>

Create an .env file in the project root:

```
RESEND_API_KEY=your_key_here
PORT=3001```
